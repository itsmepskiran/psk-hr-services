# PSK Services — HR & Business Consulting Website

A modern, single-page marketing site for PSK Services (Part of BMG Enterprises), an HR and business consulting firm.

## Stack

Plain HTML, CSS and vanilla JavaScript — no build step required.

## Structure

- `index.html` — page markup
- `style.css` — styling (palette matches the PSK shield logo: navy `#0B2C4D`, green `#1F9D55`, gold `#F5B301`)
- `script.js` — nav toggle, scroll header, contact form handling
- `assets/logo.svg` — emblem recreation used as favicon and in the header/footer

## Run locally

Open `index.html` directly in a browser, or serve it:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

Enabled via GitHub Pages from the `main` branch.
